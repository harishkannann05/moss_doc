# Moss Landing Page

A premium landing page for the Moss project, inspired by modern developer platforms like MongoDB.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS (v3) + Framer Motion
- **Icons**: Lucide React
- **Design**: Glassmorphism, Dark Mode, Bento Grids

## Getting Started

1. Navigate to the project directory:
   ```bash
   cd frontend
   ```

2. Install dependencies (if not already):
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

- `src/components/`: Reusable UI components (Hero, Features, Navbar, etc.)
- `src/App.jsx`: Main application layout
- `src/index.css`: Global styles and Tailwind directives

## Designed For

High-performance developer experience and premium aesthetics.
